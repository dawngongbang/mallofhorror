# 몰오브호러 온라인 (Mall of Horror Online)
## 프로젝트 기획서 v2.0

> 작성일: 2026-03-23
> 목적: 몰오브호러 보드게임을 서버 없이 친구들과 온라인으로 플레이하기 위한 웹 앱 구현

---

## 1. 게임 개요

몰오브호러는 3~6인 협상/배신 보드게임이다.
쇼핑몰을 배경으로 각 플레이어가 동일한 캐릭터 세트를 조종하며,
매 라운드 좀비의 습격을 피해 살아남고 최고 점수를 획득하는 것이 목표다.

---

## 2. 맵 구성

주차장(4번)이 중앙에 있고 나머지 5개 구역이 오각형 형태로 배치.
**이동 시 배치는 관계없음 — 모든 구역 간 자유 이동.**

```
           [5. 보안실]
          (수용3/방어3)

[6. 슈퍼마켓]         [1. 화장실]
 (수용6/방어4)         (수용3/방어3)

        [4. 주차장]
       (수용∞/방어0)

[3. 장난감가게]       [2. 옷가게]
  (수용4/방어4)        (수용4/방어4)
```

### 구역별 특성

| 번호 | 구역 | 최대 수용 | 방어 한도 | 특수 이벤트 |
|------|------|----------|----------|-----------|
| 1 | 화장실 | 3명 | 3 | 없음 |
| 2 | 옷가게 | 4명 | 4 | 없음 |
| 3 | 장난감가게 | 4명 | 4 | 없음 |
| 4 | 주차장 | 무제한 | 0 | 아이템 탐색 투표 |
| 5 | 보안실 | 3명 | 3 | 보안관 투표 |
| 6 | 슈퍼마켓 | 6명 | 4 | 없음 |

> **방어 한도**: 캐릭터가 아무리 많아도 이 수치를 초과한 방어는 불가.
> **주차장**: 방어 한도 0 → 좀비가 1마리라도 있으면 항상 습격 발생.

---

## 3. 캐릭터

각 플레이어는 동일한 캐릭터 세트를 소유하고 시작한다.

| 캐릭터 | 점수 | 방어력 | 투표권 | 특성 |
|--------|------|--------|--------|------|
| 미녀(Belle) | 7점 | 1 | 1 | 미녀 집결 구역 판정 기준 |
| 터프가이(Tough Guy) | 5점 | **2** | 1 | 좀비 2마리 방어 |
| 건맨(Gunman) | 3점 | 1 | **2** | 투표권 2개 |
| 아이(Kid) | 1점 | 1 | 1 | — |

**플레이어 수별 캐릭터 세트:**
- 3인: 미녀 + 터프가이 + 건맨 + 아이 (인당 4캐릭터)
- 4~6인: 미녀 + 터프가이 + 건맨 (인당 3캐릭터)

### 방어력 계산
```
구역 방어력 = min(구역 방어 한도, 구역 내 캐릭터 방어력 합산 + 아이템 보너스)

예) 슈퍼마켓에 터프가이(2) + 미녀(1) + 미녀(1) + 건맨(1) = 합산 5
    min(5, 4) = 4 → 좀비 5마리면 뚫림
```

## 4. 아이템

게임 시작 시 전체 21장의 아이템 덱에서 각 플레이어에게 3장씩 랜덤 지급.

| 아이템 | 장수 | 효과 |
|--------|------|------|
| 히든카드 | 3 | 미정 |
| 보안카메라 | 3 | 주사위 결과 확인 가능 (보안관과 동일) |
| 스프린트 | 3 | 미정 |
| 협박 | 3 | 투표권 +1 (1회성) |
| 하드웨어 | 3 | 습격 판정 시 방어력 +1 |
| 도끼 | 1 | 좀비 1마리 제거 |
| 권총 | 1 | 좀비 1마리 제거 |
| 샷건 | 1 | 좀비 2마리 제거 |
| 야구배트 | 1 | 좀비 1마리 제거 |
| 수류탄 | 1 | 좀비 2마리 제거 |
| 전기톱 | 1 | 좀비 2마리 제거 |

**총 21장** (일반 5종×3 + 무기 6종×1)

## 5. 게임 흐름

### 5.1 초기 세팅

```
1. 각 플레이어에게 아이템 3장 랜덤 지급
2. 보안관 랜덤 선정 (임시 보안관)
3. 플레이어 순서 확정 (게임 내내 유지)
4. 캐릭터 배치 페이즈:
   - 보안관부터 순서대로 주사위 2개 굴림
   - 나온 두 구역 번호 중 하나 선택 → 자기 캐릭터 1개 배치
   - 해당 구역이 가득 찼으면 원하는 곳에 자유 배치
   - 모든 플레이어가 캐릭터를 다 배치할 때까지 반복
5. 세팅 완료
```

### 5.2 첫 라운드 시작

```
첫 라운드는 임시 보안관만 있음 (보안실 점유 시 진짜 보안관)
→ 주사위 4개 굴리기만 (좀비 배치) → 이동 없이 바로 이벤트 페이즈
```

### 5.3 일반 라운드

```
[1단계] 주사위 굴리기
  - 보안관이 주사위 4개 굴림
  - 진짜 보안관(보안실 점유): 결과 혼자 확인
    → 공개 / 거짓말 / 침묵 자유
  - 보안카메라 아이템 사용자도 확인 가능
  - 임시 보안관: 공개

[2단계] 이동 페이즈
  ① 보안관부터 순서대로 이동할 캐릭터 공개 선언 (본인 소유만)
  ② 모든 플레이어가 목적지 구역 번호 비공개 선택
  ③ 동시 공개
  ④ 보안관 순서대로 이동 처리
     - 같은 구역으로는 이동 불가 (반드시 다른 구역)
     - 목적지가 가득 찼으면 → 주차장으로 튕겨남

[3단계] 이벤트 페이즈 (1번 구역부터 6번까지 순서대로)
  A. 주사위 결과 공개 + 좀비 배치
  B. 추가 좀비 배치:
     - 미녀가 가장 많은 구역 +1 (동률이면 스킵)
     - 캐릭터가 가장 많은 구역 +1 (동률이면 스킵)
  C. 각 구역 처리 (1번→2번→3번→4번→5번→6번):
     - 좀비 수 > 방어력? → 습격 발생
     - 아니면 안전 (좀비 누적 유지)

[4단계] 승리 조건 체크
```

### 5.4 습격 처리

```
습격 발생 시:
  1. 해당 구역 플레이어들끼리 투표
     (투표권: 캐릭터당 1표, 건맨은 2표, 협박카드 사용 시 +1표)
  2. 가장 많이 지목된 플레이어의 캐릭터 사망
     - 해당 구역에 캐릭터가 1개면 그 캐릭터 사망
     - 여러 개면 플레이어가 선택
  3. 사망 발생 → 해당 구역 좀비 전부 소멸

동률 처리:
  1차 동률 → 재투표
  2차 동률 → 해당 구역 밖 전체 플레이어도 참여
  이후 동률 → 동률 깨질 때까지 계속 투표

주차장 특수:
  - normal 모드: 위와 동일 (투표로 1명 사망)
  - hardcore 모드: 좀비 수만큼 사망자 발생 (투표로 순서 결정)
```

### 5.5 특수 구역 이벤트

```
4번 주차장 (해당 구역에 캐릭터 있을 때):
  습격 처리 완료 후 → 아이템 탐색자 투표
  → 당선자: 남은 아이템 덱에서 3개 랜덤 확인 (본인만)
  → 1개 소유 / 1개 아무 플레이어에게 증정 / 1개 반납

5번 보안실 (해당 구역에 캐릭터 있을 때):
  습격 처리 완료 후 → 보안관 투표
  → 동률 2회까지: 기존 보안관 유지
  → 당선자: 다음 라운드부터 진짜 보안관
  → 보안관은 주사위 결과를 혼자 확인 + 공개 여부 자유
```

### 5.6 보안관 시스템

```
임시 보안관: 랜덤 선정, 주사위 결과 공개
진짜 보안관: 보안실 투표로 선출, 보안실에 캐릭터가 있어야 유지
  - 보안실 캐릭터 없으면 → 다시 임시 보안관 권한만

보안관 교체 시: 다음 라운드부터 턴 순서 재편
  (기존 플레이어 상대 순서 유지, 새 보안관이 1번)
```

### 5.7 승리 조건

```
매 라운드 종료 시 체크:
  전체 생존 캐릭터 수 ≤ 4
  AND 주차장을 제외한 한 구역에 모든 생존 캐릭터가 집결
  → 게임 종료

생존 캐릭터 점수 합산:
  미녀 7점 / 터프가이 5점 / 건맨 3점 / 아이 1점
  → 합산 점수 1등 플레이어 승리 (동점 시 공동 우승)
```

## 6. 기술 스택
